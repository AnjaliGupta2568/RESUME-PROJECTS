## Overview
End-to-end AI-powered traffic monitoring system using YOLOv8 for vehicle detection, OCR for number plate recognition, and real-time dashboard analytics.

## Features
- Multi-class Vehicle Detection (Car, Bus, Truck)
- Speed Estimation
- Line Crossing Detection
- Automatic Vehicle Counting
- OCR-based Number Plate Recognition
- Real-time Streamlit Dashboard

## Tech Stack
Python, YOLOv8, OpenCV, Tesseract OCR, Streamlit

## Performance
- Detection Accuracy: ~92%
- Latency Reduced by ~30%

## Run Locally

pip install -r requirements.txt
streamlit run app.py
